Installation

We need a couple of applications to run the Quiety to our local:

Node.js:

It is used to run the build processes. Download from the NodeJs from the official site and install it on your device according to your Operating System (nodejs.org)

Check NodeJs Version:

  • Open the Terminal
  • Run: node -v or node --version
  • If you get the version then the installation of NodeJs is done
$ node --version
$ v20.9.0

Next Packages Install:

Go to project directory then run this command in command line, it will install all packages that we used this template. Make sure that there is no error.


$ npm install

Go to project directory then run this command in command line,The script will process all SASS files and generate the corresponding CSS. Make sure that there is no error.


$ npm run sass

After that your installation process is done.

How to Run?

After installed all packages, you just need to run this script, it will run on your local server on port 'http://localhost:5173/'.


$ npm run dev

How to Build?

As a next.js or react project Vercel is recommended for deployment and If you want to deploy this project/template on Vercel then your package.json file will look like,

 
 "scripts": {
    "dev": "vite",
    "build": "vite build",
    "lint": "eslint .",
    "preview": "vite preview",
    "sass": "sass src/assets/scss:src/assets/css --watch --no-source-map"
  },

Now you just need to push this project to your GitHub then connect GitHub with vercel account. Then just create new project on vercel by importing from your GitHub, then deploy.


 And if you want to generate static data of this project to deploy cPanel or other hosting service then your package.json file will look like,

 
 "scripts": {
    "dev": "vite",
    "build": "vite build",
    "lint": "eslint .",
    "sass": "sass src/assets/scss:src/assets/css --watch --no-source-map"
  },

now just run npm run build, it will generate all static pages in dist folder, then just import that folder to your cPanel to deploy on production.



Folder structure


  • node_modules - Node dependencies libraries
  • package.json - All used npm packages.
  • public - All used static images.
  • src/components - contains all the template-related UI components.
  • src/components/layout - contains project layout section for all pages and components, also header and footer is included in this folder.
  • src/App.jsx- Here you will find all pages that we use in this project.
  • src/assets- contain bootstrap and custom CSS and font.

Template Customization

Change Favicon
go to index.html file

<head>
    <meta charset="UTF-8" />
    <link rel="icon" type="image/png" href="/img/favicon.png" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>

Change Title

go to src/components/PageMeta.jsx

just change pageTitle

<PageMeta title="Quiety - Software &amp; IT Solutions HTML Template" />

now  change specific page for specific name 

How to change template style?

For changing template default style, just go to src/assets/scss/main.scss file then change as per your need.

then -- npm run sass
